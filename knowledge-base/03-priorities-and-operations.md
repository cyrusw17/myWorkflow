## Priority order (validated)

1. Business name + domain
2. MVP website (credibility for outreach)
3. **Get name out there** (revenue only happens when people know you exist)

## Launch playbook (`first-steps.md`)

Work top to bottom:

1. **Go live** — DNS, deploy, SSL, form, email
2. **Founding landing** — live at groundwork-web.com (mvp-website IS the landing page)
3. **Outreach** — 10 touches/week before ads
4. **Ad test** — $50–150 only after site + form work; landing page URL, not homepage
5. **Close** — discovery → agreement → first month upfront → deliver

---

## What NOT to do right now

- Register LLCs, buy courses, build internal tools before clients
- Expand to unrelated niches
- Perfect brand guidelines before first sale
- Run ads before production form works
- Spend $500 on ads before 10 organic outreach attempts
- Sign all 3 founding clients same week
- Keep redesigning instead of selling
- Quit Discount Tire

---

## Anti-patterns

| Trap | Redirect to |
|------|-------------|
| Naming research for weeks | Ship with Groundwork Web |
| 4th site rebuild | Deploy MVP, iterate from outreach feedback |
| New framework | Use current PHP/Laravel stack |
| "Outreach when site is perfect" | Start conversations now |
| New side project | `priorities.md` current phase |
